## 使用规则

- 新增固定 UI 文案前，先补充本表和 `i18n/i18n.js`。
- 不把用户生成内容、商家输入内容或真实商品名强行改成英文。
- 英文优先使用北美生活服务 App 常用写法，避免逐字直译。
- 首页说明、空状态、加载状态和引导文案同样属于固定 UI 文案，必须从词表取词，不能留下中英混排。
