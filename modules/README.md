# 乐生活模块边界

每个模块使用独立的 `index.html`、样式与脚本，只通过 Supabase 数据表、当前登录会话和 URL 参数通信。

| 模块 | 地址 | 责任 |
| --- | --- | --- |
| 社区主程序 | `/` | 首页内容、发布、消息、个人主页、商家主页、站内管理 |
| 今日省钱 | `/deals/` | 每日价格缓存、搜索、热榜、用户爆料、商家优惠提交 |
| 租车预约 | `/rental/` | 客户选车、筛选、预约与付款流程 |
| 租车管理 | `/rental/manage/` | 车辆、服务、交接、任务与预约管理 |
| 点餐与外卖 | `/order/` | 餐桌扫码点餐、外卖、购物车与订单状态 |
| 扫码排队 | `/queue/` | 取号、预点菜、入座转订单 |
| 商家管理 | `/merchant/manage/` | 商家资料、功能开通与运营入口 |

新增功能必须先确定所属模块；不能把页面、样式或业务函数直接追加回社区主程序。
