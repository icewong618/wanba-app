# v5.500 - Modular App Build

本版本完成主程序模块化收口。

## 完成内容

- 将入口页面、全站样式和主控制器拆分为独立文件。
- 保持 `index.html` 与 `app.html` 完全一致。
- 保留并接入现有省钱、活动、消息、餐饮、租车、二手车、商家和订单独立模块。
- 重做 GitHub Pages 发布构建：完整复制所有模块目录，避免线上页面缺少脚本或样式。
- 增加发布前检查：检查版本号、入口页、脚本语法和最终发布包引用。

## 验证

- `npm run check:release`
- 入口页一致性检查
- 主控制器语法检查
- 发布目录完整性检查

## 说明

这是结构与发布稳定性版本，不刻意改变现有功能或视觉效果。推送后需要在网页、iPhone/iPad 和 Android App 各完成一轮核心路径回归。
