# 乐生活 v5.160

## 发布二级分类

- 发布笔记时，选择美食、玩乐、好物、生活或社区后，可选对应的细分分类。
- 二级分类为可选项，不会阻碍普通内容发布。
- 信息流封面、笔记详情和搜索结果显示“主分类 · 二级分类”。
- 支持编辑已发布的新笔记、恢复本机草稿后重新选择二级分类。
- 社区支持二手交易、房屋出租、求职招聘、拼车搭子、失物招领和邻里互助，为后续审核与推荐准备数据。
- 首页自动刷新调整为仅在页面可见、停留首页时每分钟进行一次，降低偶发网络中断的重试频率。

## 数据库

已新增 `posts.subcategory` 与对应查询索引。旧笔记保持为空，不受影响。

## 验收

1. 打开发布页，选主分类后确认出现对应二级分类。
2. 不选二级分类仍可发布。
3. 发布“社区 · 二手交易”后，在首页封面、详情和搜索结果确认路径展示正确。
4. 编辑该笔记或从草稿箱恢复，确认二级分类可以保留或更改。
