# 乐生活 5.104 - 会员权益与管理收口版

## 本版内容

- 出示会员卡页面将“已领取优惠券”放到“最近使用记录”上方。
- 商家会员管理页分为“近期新加入会员”和“最近消费会员”。
- 商家完成消费、积分、奖励兑换或优惠券核销后，会员会在消息页的“会员权益”看到对应通知。
- 会员通知可直接打开对应的会员卡或优惠券。
- 从商家会员详情打开用户主页时，用户主页改为全屏安全区页面；关闭后会回到此前的会员详情。

## 数据库

`Supabase-member-activity-notifications-5.104.sql` 已部署到线上数据库，用于保存会员权益通知，无需再次手动执行。

## 建议验收

1. 用户打开“出示会员卡”，确认已领取优惠券在最近使用记录之前。
2. 商家进入会员管理，确认能看到近期加入和最近消费两组会员。
3. 商家给会员增加消费次数或积分后，用户在消息的“会员权益”查看通知。
4. 商家点击会员头像打开用户主页，再关闭，确认返回会员详情页且顶部按钮可点击。
