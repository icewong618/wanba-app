# v5.245 Merchant Feature Center

## Included

- Added the full-screen Merchant Feature Center from the merchant owner menu.
- Added the first five modules: table ordering, takeout ordering and pickup number, booking, event registration, and ticketing.
- Added five roadmap modules: queue, quote, attendance, presale/group buy, and delivery tracking.
- Removed the default table-order management card from the merchant home page.
- Modules that require a phone number cannot be added until the merchant profile has one.

## Current Scope

This release creates the feature-center interface and local feature selection state only. Each module's real business workflow, database schema, and customer-facing entry will be implemented one at a time.

## Future Paid Plans

- Merchant subscriptions will define which modules a merchant may enable.
- A merchant will only see the modules included in its current plan as available to add.
- Upgrades will unlock additional modules without changing the merchant page layout.
- Pricing, billing, plan changes, and payment are intentionally out of scope for this release.
