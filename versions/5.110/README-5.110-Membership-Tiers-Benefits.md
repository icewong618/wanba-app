# 乐生活 5.110 - 会员等级与权益规则第一版

## 商家可配置

- 每次消费自动赠送的积分。
- 集章兑换目标和奖励。
- 银卡、金卡、黑金会员的积分门槛。
- 生日月权益文案，例如双倍积分或赠送小食。

## 用户与商家体验

- 商家确认一次消费时，会同时累计消费次数和设定的积分，并写入使用记录。
- 用户会员卡与出示会员卡页面展示当前等级、积分、下一等级进度和生日月权益。
- 商家会员管理列表与会员详情显示会员等级；本月生日会员会有提示。
- 用户会在“会员权益”收到本次消费次数和积分的合并通知。

## 验收

1. 商家打开“编辑商家资料”，设置积分、等级门槛和生日月权益后保存。
2. 商家扫码或输入会员号，选择“消费 +1”，确认消费次数和积分同时增加。
3. 用户打开会员卡，确认等级和距下一等级的积分显示正确。
4. 把测试用户出生月份设为当前月份，确认会员卡和商家会员详情显示生日月权益。
