# 乐生活 5.106 - 公开用户主页统一版

## 本版内容

- 点击其他普通用户头像后，公开主页改为与“我”页面一致的全屏封面布局。
- 背景图延伸至顶部，所有顶部操作都在安全区域内。
- 公开主页取消“用户主页”标题，左侧为菜单，右侧为关闭按钮。
- 菜单预留后续项目，当前提供“发私信”。原本的分享入口改为发私信。
- 用户名、ID、IP属地、简介、年龄和标签均统一显示在背景图上。
- 新增 `profiles.ip_location`，只保存城市／国家级 IP 属地文本，不保存真实 IP 地址。

## 验收

1. 从首页、消息或会员管理点击普通用户头像，确认进入全屏背景主页。
2. 确认右上角关闭、左上角菜单在 App 刘海屏安全区内可点击。
3. 点击“发私信”，确认进入消息页并打开与该用户的对话。
4. 用户重新打开网站后，自己的 IP 属地会同步到公开主页。
