# 乐生活 v5.159

## 商家数据中心与本机草稿箱

- 商家发布面板新增数据中心，可查看近 7 天或近 30 天的内容浏览、互动、收藏、优惠券、会员与签到数据。
- 数据中心只使用商家有权限读取的真实业务记录；无法读取的指标显示为“—”，不会伪装成 0。
- 普通用户发布页新增本机草稿箱。输入、选择分类、上传图片、制作文字配图等操作会自动保存。
- 草稿可继续编辑或删除，最多保留 12 篇。
- 草稿和图片只保存在当前设备，不上传服务器；清除浏览器数据、卸载 App 或更换设备后会消失。
- 发布成功后会自动删除对应草稿。

## 验收

1. 认证商家进入发布面板，点击“数据中心”，切换近 7 天与近 30 天。
2. 普通用户填写标题和正文后关闭发布页，再打开“草稿箱”继续编辑。
3. 在草稿箱删除草稿，确认其不再出现。
4. 发布草稿内容后，确认草稿箱自动移除该草稿。
