# v5.166 分享卡与首次兴趣选择版

## 分享卡

- 公开笔记链接会输出笔记封面、标题和正文摘要；文字超过分享卡常见长度时自动以省略号截短。
- 公开用户主页链接会输出头像、用户名和个人简介。
- 商家微网站链接优先使用商家头像作为缩略图，介绍作为摘要。
- 网页和 App 内的分享面板可直接生成、复制或发送上述链接；Facebook、微信等外部平台由 Cloudflare Worker 提供动态 Open Graph 信息。

## 首次兴趣选择

- 首次进入可多选兴趣：美食、咖啡、亲子、活动、户外、旅行、摄影、二次元、二手、租房、交友、求职、省钱、数码、家居、健康等。
- 推荐排序会参考兴趣、关注关系、互动和发布时间；“最新”仍严格按发布时间排列。
- 兴趣选择仅保存在当前设备，可随时从首页“偏好”修改。

## 发布

- 无需运行 SQL。
- 推送后等待 `Deploy to GitHub Pages` 和 `Deploy Share Card Worker` 都变为绿色。
