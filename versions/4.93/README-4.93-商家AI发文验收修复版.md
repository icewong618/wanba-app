# 乐生活 v4.93 商家 AI 发文验收修复版

本版本是在 v4.92 商家 AI 发文补完版基础上的验收修复版，重点修复商家 AI 发文流程里编辑内容的保存稳定性。

## 本版更新

- 商家 AI 预览页切换平台前，会先保存当前标题、正文和标签。
- 切换旧版本/新版本前，会先保存当前正在编辑的内容。
- 点击“下一步：选择平台”前，会先保存当前编辑内容。
- 点击复制标题、复制正文、复制已选平台文案前，会先同步当前输入框内容。
- 点击发布到乐生活前，会先同步当前输入框内容。
- 预览页增加发布说明，明确：
  - 乐生活站内发布使用“乐生活”标签页内容。
  - 外部平台按所选平台生成可复制文案。
  - AI 只辅助整理文案，不生成价格。
  - 视频 AI 暂未开放。

## 数据库

本版本不需要新增 SQL。

## 部署

- 首页文件：`index.html`
- 版本归档：`versions/4.93/`
- GitHub Actions 每日价格刷新路径：`versions/4.93`
