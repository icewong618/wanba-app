# 乐生活公测前验收清单

版本：v5.510 起

每一项须在网页、iPhone/iPad 和 Android 三端记录结果；发现问题时附截图、设备型号、系统版本、页面路径和复现步骤。

## 自动检查

- [ ] GitHub Actions `Deploy to GitHub Pages` 为绿色。
- [ ] GitHub Actions `Public Beta Smoke Check` 为绿色。
- [ ] 首页、App 入口、省钱、本周、消息、商家分享卡和线上版本号均可读取。
- [ ] `npm run check:release` 通过。

## 账号与社区

- [ ] 新用户用邮箱和密码注册，收到确认邮件并可完成验证。
- [ ] 已注册用户登录、退出、保持登录、错误密码提示均正常。
- [ ] 忘记密码邮件可收到、链接可用、改密后可登录。
- [ ] 发布纯文本、单图、多图（最多 15 张）、文字配图、YouTube/TikTok 链接均正常。
- [ ] 编辑笔记后封面、正文、图片顺序和作者头像会更新。
- [ ] 笔记详情可滚动，点赞、收藏、评论、删除自己的评论和转发均正常。
- [ ] 消息中的评论和 @、新增关注、赞和收藏能看到最新昵称和头像。

## 个人与商家

- [ ] 个人主页资料、背景、头像、简介换行、标签、关注和粉丝在重新登录后仍保留。
- [ ] 会员卡、积分、奖励、消费记录、已领取优惠券和二维码可正常显示。
- [ ] 商家主页信息、营业时间、地图、电话、外链、店铺、动态和优惠券正常。
- [ ] 商家员工权限与主号功能边界正确，未授权操作会给出提示。

## 餐饮与订单

- [ ] 餐桌二维码打开点餐页；浏览器可点餐，优惠券提示安装 App。
- [ ] 外卖二维码、排队二维码和 App 深链分别进入正确页面。
- [ ] 加菜、催菜、上菜、后厨完成、收银完成订单和订单历史正常。
- [ ] 优惠券按支付方式、金额、日期和时段正确显示、核销、抵扣。

## 租车与二手车

- [ ] 租车搜索筛选、车辆详情、附加服务、保险、预约、付款准备流程正常。
- [ ] 租车商家后台可查看车辆状态、预约、交车还车、清洁维修任务。
- [ ] 二手车浏览、筛选、收藏、对比、试驾、卖车估价和 VIN 解析正常。

## 跨端与异常网络

- [ ] 网页、iPhone、iPad、Android 的首页首屏无遮挡，底部导航一直可点。
- [ ] 各全屏页的关闭、返回和三点菜单都回到进入前页面。
- [ ] iPhone 刘海屏、iPad、安卓全面屏和折叠屏的顶部/底部安全区正常。
- [ ] 弱网下显示加载或重试，不出现白屏；无网时显示“没有网络连接”页面。
- [ ] 从二维码或深链打开已安装 App 时，进入对应功能而非首页或外部浏览器。
- [ ] 下拉刷新、重新进入 App、缓存更新后能显示最新版本与内容。

## 通过标准

公测前不得存在 P0（无法进入首页、无法登录、数据丢失、白屏、无法发帖）或 P1（主要功能不能完成、跨端核心导航失效）问题。其余体验问题统一记录后排入公测修复版本。
