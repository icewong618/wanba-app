# 乐生活 v5.249 - 优惠券规则与结算方式增强

本版更新：

- 客户扫码点餐页支持在顶部下拉刷新已点菜品，不再固定自动刷新。
- 结算方式拆分为现金、刷卡、在线支付。
- 小费统一更名，在线支付支持比例小费与自定义金额。
- 优惠券可配置适用星期、每日时段和支付方式。
- 顾客领取、出示、点餐结算和商家扫码核销均会检查适用规则。
- 商家扫码时会明确显示优惠券不能使用的原因。

数据库迁移：`Supabase-coupon-rules-5.249.sql` 已在项目数据库应用。

说明：在线支付目前仅展示结算流程，尚未接入真实支付渠道，因此不会伪造支付成功或提前核销优惠券。接入支付后，应由支付成功的服务器端回调在同一事务内核销已选优惠券。
