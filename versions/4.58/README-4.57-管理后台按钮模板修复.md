# 乐生活 4.57 管理后台按钮模板修复

## 修复内容

修复管理后台左上角按钮显示 `${uiIcon('bag',13)}` 的问题。

原因是该位置在静态 HTML 中写入了模板字符串表达式，浏览器不会执行它，所以原样显示在页面上。

本版将两个静态按钮改为直接写入 SVG 图标：

- 管理后台“审核优惠”
- 个人页“成为商家”

## 数据库

本版不需要执行新的 Supabase SQL。

## 验收

1. 启动页显示 `v4.57`。
2. 管理后台左上角显示图标 + “审核优惠”，不再显示 `${uiIcon(...)}`。
3. 个人页“成为商家”按钮也显示正常图标。
