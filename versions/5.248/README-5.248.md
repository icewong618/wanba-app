# 乐生活 v5.248 - 扫码点餐流程增强

## 本版内容

- 餐桌二维码管理支持删除未产生订单的餐桌；已有订单记录的餐桌保留历史并提示暂停使用。
- 商家订单管理页打开期间每 3 分钟自动刷新，也保留手动刷新。
- 新加菜按批次分段显示，订单内每个菜品可标记“上桌”；误点可撤回。
- 顾客点餐页展示已点内容与上桌状态，全部上桌后显示结算入口。
- 结算页支持现金/刷卡与在线支付两套界面：已领取优惠券可按商家设置选择单张或多张；在线支付只展示 Apple Pay、Google Pay、信用卡、商家礼品卡和消费选项，暂不真实扣款。
- 店长可设置优惠券单张/叠加规则及默认消费百分比。

## 数据库

`Supabase-merchant-order-enhancement-5.248.sql` 已部署到生产数据库，文件保留作版本记录。
