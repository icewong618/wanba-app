# 乐生活 5.105 - 信息流稳定与会员回执版

## 本版内容

- 首页只读取公开帖子，并减少单次读取数量。
- 为公开信息流新增数据库部分索引，降低帖子读取的 statement timeout 风险。
- 首页请求加入超时保护和更轻量的备用请求，发生网络或数据库短暂拥堵时更快回退。
- 奖励兑换和撤销兑换后，商家操作页会显示结果回执。
- 撤销奖励兑换时，用户会在“消息 - 会员权益”收到消费次数恢复通知。

## 数据库

`Supabase-public-feed-performance-5.105.sql` 需要部署到线上数据库。本次发布会由开发环境直接完成部署，无需再手动运行。

## 建议验收

1. 刷新首页，确认帖子正常出现，控制台不再出现 posts 的 statement timeout。
2. 商家兑换会员奖励后，确认用户收到“会员奖励已兑换”消息。
3. 商家撤销奖励兑换后，确认用户收到“会员奖励兑换已撤销”消息，会员卡次数恢复。
4. 打开会员卡，确认优惠券仍在近期使用记录上方。
