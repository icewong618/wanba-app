# 乐生活 v5.240 - Merchant Standard Toolkit

## 商家开通即有

- 店铺主页：Logo、封面、介绍、营业时间、地址地图、电话和外链。
- 内容动态：普通笔记、置顶内容、YouTube/TikTok 链接、商家公告。
- 联系咨询：站内私信、电话、导航和外部社媒入口。
- 商家工具：AI 发文、手动发文、多平台文案复制与系统分享、优惠提交和数据中心。
- 团队账号：店长为商家主号，可邀请或撤销店员和运营矩阵账号。

## 说明

- 店长拥有商家资料、团队与账户的最终控制权。
- 店员和运营使用各自的乐生活账号，不共享商家密码。
- Facebook、Instagram、Google Business 的自动发布需要各商家完成官方 OAuth 授权；本版本已提供多平台文案与分享入口，但不会伪装成自动发布。
