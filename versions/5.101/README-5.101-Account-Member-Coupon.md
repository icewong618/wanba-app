# 5.101 账号、会员与优惠券体验版

- 登录改为明确的“登入 / 新用户注册”入口；密码错误不再被当成首次注册。
- 增加忘记密码邮件和显示密码选项；重置链接可设置新密码。
- 个人资料、公开主页、粉丝和会员列表优先显示 profiles 中的最新头像、昵称、背景、简介、标签、性别和年龄。
- “我的”页新增可点击的关注与粉丝数量，全屏查看关注/粉丝，并支持回关与互关状态。
- 新增粉丝详情页增加顶部安全距离。
- 商家会员管理改为扫码或查找后自动进入全屏会员操作页；列表只显示近期加入的 10 位会员。
- 会员记录首页显示 10 条并可按最近三天、本周、本月、全部查看。
- 优惠券领取改为数据库保存；用户可出示优惠券二维码，商家可扫码核销。
- 商家和个人介绍限制 50 字，移动端最多显示 3 行。
- `Supabase-coupon-claims-5.101.sql` 已部署到 Supabase，不需要手动运行。
