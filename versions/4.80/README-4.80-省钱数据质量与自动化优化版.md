# 乐生活 v4.80 省钱数据质量与自动化优化版

## 本版目标

在 4.70 省钱运营后台基础上，提升省钱页的数据质量、排序逻辑和食品低价比较能力。

## 新增与优化

- 今日爆款 / 省钱列表排序优化：
  - 人工确认优先
  - 热门优先
  - 有真实价格优先
  - 需核对和过期内容靠后
  - 最近更新优先
- 食品低价区增加单位价格提示：
  - 牛奶显示每加仑
  - 大米显示每磅
  - 鸡蛋显示每枚
- 管理后台“省钱运营”增强：
  - 已过期内容更醒目
  - 需核对内容优先显示
  - 显示单位价格
  - 新增“标记过期”
  - 新增“恢复显示”

## 数据库

本版不需要新增 SQL。

前提：已执行 4.70 的 `Supabase-deal-price-ops-4.70.sql`。
