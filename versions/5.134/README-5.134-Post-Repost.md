# 5.134 笔记站内转发版

## 新功能

- 笔记分享面板新增“转发笔记”。
- 转发采用全屏编辑页，可附加不超过 300 字的感想。
- 发布后生成引用原笔记的卡片，点击可回到原笔记。
- 原帖标题、作者、封面与摘要会保存为快照；原帖后续删除时，转发笔记仍可正常展示。

## 上线前必做

在 Supabase SQL Editor 完整运行同版本文件：

`Supabase-post-reposts-5.134.sql`

未运行前，首页仍可正常加载，但站内转发会提示先完成数据库升级。

## 验收

1. 打开任意笔记，点右上角分享。
2. 在分享面板选择“转发笔记”。
3. 输入一段感想并发布。
4. 首页、个人主页与笔记详情均应显示引用卡片。
5. 点击引用卡片，应返回原笔记。
