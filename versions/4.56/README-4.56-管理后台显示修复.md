# 乐生活 4.56 管理后台显示修复

## 修复内容

4.56 修复管理后台打开后一片空白的问题。

根因是 `page-admin` 节点被嵌套进了 `page-profile` 里面。页面切换时，系统会先隐藏所有 `.page`，导致管理后台本身虽然被设置为显示，但它的父级“我的页面”仍然隐藏，所以用户看到空白。

本版新增 `ensureAdminPageAtRoot()`：

- 切换页面前检查 `page-admin` 是否被嵌套在 `page-profile` 内。
- 如果嵌套，则自动移动到页面顶层、底部导航之前。
- 管理后台之后可正常显示权限提示、加载状态、用户、帖子、评论和封禁记录。

## 数据库

本版不需要执行新的 Supabase SQL。

如果管理后台能打开但提示没有权限，请确认当前账号已加入 `deal_admins` 表。

## 验收

1. 启动页显示 `v4.56`。
2. 点击“管理后台”后不再空白。
3. 未登录时显示权限/登录提示。
4. 管理员账号登录后显示注册用户、近期发帖、评论回复和封禁记录。
