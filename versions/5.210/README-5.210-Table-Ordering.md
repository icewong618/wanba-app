# 5.210 商家餐桌扫码点餐第一版

## 已完成

- 商家创建、暂停餐桌，并得到每张桌子的专属二维码和点餐链接。
- 顾客登录后扫码进入点餐页；可选择桌位、按分类点菜、提交订单并继续加菜。
- 订单由数据库重新读取商品价格与上下架状态，前端不能自行修改价格。
- 商家可查看待确认、制作中、已上桌、已完成的订单，并更新订单状态。
- 商家矩阵账号中的运营、店员也可进入订单管理。
- 订单只作为到店点餐与处理记录；本版不包含线上支付。

## 需要测试

1. 商家进入“我的”页，点击“管理扫码点餐”，新建桌位。
2. 用“复制链接”在已登录的浏览器打开，确认进入指定桌位菜单。
3. 在商品编辑页确认“可在扫码点餐菜单中点选”已启用，并为商品填写价格。
4. 顾客提交订单后，商家在订单管理中确认接单、开始制作、标记上桌与完成。
5. 顾客再选择菜品提交，确认同一订单显示“加菜”。

## 本版边界

- 顾客需要登录，便于防止匿名伪造、支持后续加菜和订单追踪。
- 不含付款、库存扣减、厨房打印和通知推送；这些留给下一阶段接入。
