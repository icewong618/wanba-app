# 乐生活 v5.250 - 扫码点餐账单与优惠券 App 限制

本版更新：

- 网页端优惠券领取、出示与结算改为 App 下载引导；网页仍可扫码点餐。
- 优惠券支持现金、刷卡/在线支付两类适用支付方式。
- 商家订单页保留处理中、已上桌、催菜中、已完成四个分类。
- 顾客可催菜；商家端新加菜按最新点菜时间重新排序。
- 餐桌删除即时生效；有历史订单的餐桌会安全停用并从可扫码列表移除。
- 商家以现金或刷卡完成订单后生成消费账单，顾客可在个人页面的“消费记录”查看。
- 优惠券星期改为两行横向排布；小费设置可正常输入英文逗号。

真实在线支付尚未接入。本版不伪造支付成功；接入支付后，支付渠道成功回调应调用订单完成与账单函数，并在同一服务端流程中核销优惠券。
