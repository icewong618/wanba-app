# 乐生活 v4.99b - App 封装适配与个人页顶部重构

本版仍属于内部验收版本，不作为公开测试版。

## 更新内容

- 修复封装 App 后顶部、底部贴边的问题。
- 底部导航、详情页底部评论栏、弹窗增加底部手势条安全区。
- 禁止移动端双击/点击产生页面放大。
- 禁止图片默认拖拽，减少 WebView 中页面被拖偏的问题。
- 笔记详情页图片轮播改为横向滑图、纵向滚动互不干扰。
- 个人页去掉“个人资料”标题，顶部直接展示背景图。
- 个人页支持更换背景图。
- 头像、ID、IP 属地、简介、标签显示在背景图上。
- 管理后台与切换账号入口收进左上角三条杠菜单。
- 去掉“已登录：XXXX账号”的可见状态条。
- 商家主页去掉顶部账号条，直接以封面图开场。
- 个人页与商家页继续保留上一版完成的响应式笔记宫格。

## 数据库

本版不需要执行新的 Supabase SQL。
