# 乐生活 v4.70 省钱数据审核与运营后台版

## 本版目标

把省钱栏目从“前台展示真实来源”推进到“管理员可运营、可审核、可控制展示”的第一版。

## 新增功能

- 管理后台新增“省钱运营”标签。
- 管理员可查看最近的每日价格缓存：
  - 商家
  - 商品
  - 价格 / 官网入口
  - 来源类型
  - 更新时间和过期时间
  - AI 中文摘要或价格说明
- 管理员可执行：
  - 官网核对
  - 设为热门 / 取消热门
  - 标记人工确认
  - 标记需要核对
  - 隐藏前台展示
- 前台省钱卡片显示更清楚的数据来源：
  - 人工维护
  - 官方周广告
  - 官方限时优惠
  - 实验抓取
  - 固定商品页等

## 数据库

本版需要执行：

- `Supabase-deal-price-ops-4.70.sql`

执行后会给 `deal_daily_price_cache` 增加审核与展示字段，并让 `deal_current_prices` 自动过滤已隐藏内容。
