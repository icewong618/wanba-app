# 乐生活 5.120 - 附近会员卡体验版

## 本版内容

- 会员卡“附近”模式只显示已配置门店坐标的商家，并按真实距离排序。
- 第一张最近门店会显示“最近门店”标记。
- 附近模式中的第二个操作改为“到店导航”，直接打开地图路线。
- 门店未设置位置时会明确提示，不再混入附近结果造成误解。
- 用户定位仅在当前设备临时用于计算距离，不上传、不保存。

## 商家准备

商家在“编辑商家资料”中使用“查找位置”或“使用当前定位作为门店位置”，保存后即可加入附近会员卡排序。

## 验收

1. 普通用户打开“我 - 会员卡 - 附近”，允许定位权限。
2. 确认会员卡按距离排序，最近商家显示标记。
3. 点击“到店导航”，确认地图能打开对应门店。
4. 未设置坐标的商家不应显示在附近列表中。
