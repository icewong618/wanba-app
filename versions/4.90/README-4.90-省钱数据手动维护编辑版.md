# 乐生活 v4.90 省钱数据手动维护编辑版

## 本版目标

让管理员可以直接在“管理后台 → 省钱运营”里编辑每日价格缓存，不再需要进入 Supabase 表格手动改字段。

## 新增功能

- 省钱运营列表新增“编辑”按钮。
- 支持直接编辑：
  - 商品中文名
  - 商品英文名
  - 当前价格
  - 原价
  - 单位
  - 库存状态
  - 来源链接
  - 中文摘要 / 推荐理由
  - 价格说明
  - 是否食品低价
  - 是否热门
- 保存后自动：
  - 重新计算折扣百分比和省钱金额
  - 标记为“人工确认”
  - 恢复前台展示
  - 刷新更新时间和过期时间
  - 刷新前台省钱缓存

## 数据原则

AI 只整理中文摘要，不生成价格。价格必须来自管理员人工填写、官方页面核对、外部抓取或联盟来源。

## 数据库

本版不需要新增 SQL。

前提：已执行 4.70 的 `Supabase-deal-price-ops-4.70.sql`。
