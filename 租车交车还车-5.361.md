# 乐生活租车交车/还车第一版（5.361）

## 商家操作

1. 客户完成付款且商家确认预约后，在预约详情选择“办理取车交接”。
2. 填写取车里程、油量或电量、交接备注，并可上传最多 8 张车况照片。
3. 预约自动改为“租用中”。
4. 客户还车时，在同一预约选择“办理还车交接”，填写还车里程、油量或电量、车况和照片。
5. 完成还车后预约变为“已完成”，车辆自动进入“清洁中”。清洁完成后在车队看板改回“可预约”。

## 客户可见内容

客户在“我的预约”查看预约详情时，可看到取车/还车的时间、里程和油量/电量状态。

## 本版边界

- 损坏和违章费用仍在商家“收款 / 押金 / 减免”中登记。
- 里程、油量和照片均由商家操作保存；下一版可增加客户签字与车辆检查清单。
